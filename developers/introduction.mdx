## API referansı

Kanonik OpenAPI sözleşmesi yayımlandığında kullanılabilir endpoint'ler,
request ve response şemaları, kimlik doğrulama, hata modeli ve desteklenen
kabiliyetler bu geliştirici yüzeyinde listelenir.

<Warning>
  Bu sitede API referansı bulunmayan bir endpoint, payload, ortam, kota veya
  sağlayıcı kabiliyetini kullanılabilir ürün sözleşmesi olarak
  değerlendirmeyin.
</Warning>
