FaturaDev, yazılımınız ile müşterinizin yetkilendirdiği özel entegratör arasında
ortak bir geliştirici yüzeyi oluşturur. Özel entegratör hesabının ve mali
sorumlulukların yerini almaz.

## FaturaDev ne sağlar?

FaturaDev'in public API sözleşmesi, farklı özel entegratör bağlantıları için
ortak müşteri entegrasyon yüzeyini tanımlar. Bir özellik ancak bu sitede
yayımlanan ürün ve API dokümanında yer aldığında kullanılabilir kabul edilir.

FaturaDev müşterinin özel entegratör seçimini veya sağlayıcıyla yaptığı
sözleşmeyi değiştirmez. Bağlantı, müşterinin açıkça yetkilendirdiği hesap ve
kapsamla sınırlıdır.

## Özel entegratör ne sağlar?

Yetkili özel entegratör; kendi sözleşmesi ve GİB yetkisi kapsamında mali imza,
GİB iletimi, sağlayıcı hesabı ve mevzuata tabi işlemlerden sorumludur.
FaturaDev bu görevleri üstlenmez ve müşterinin mali mühür ya da özel imza
anahtarını almaz.

| Sorumluluk | FaturaDev | Yetkili özel entegratör |
| --- | --- | --- |
| Müşteri yazılımı için ortak API sözleşmesi | Evet | Sağlayıcıya özgü olabilir |
| Özel entegratör hesabı ve ticari sözleşme | Hayır | Evet |
| Mali mühür, e-imza veya XAdES | Hayır | Yetkili hizmet kapsamında |
| GİB'e doğrudan belge iletimi | Hayır | Yetkili hizmet kapsamında |
| Desteklenen operasyonların public sözleşmesi | FaturaDev API için | Sağlayıcı API'si için |

## İstek yolu

```text
Müşteri yazılımı
      ↓
FaturaDev public API
      ↓
Müşterinin yetkilendirdiği özel entegratör bağlantısı
      ↓
GİB — özel entegratörün sorumluluk alanı
```

Bu ayrım, FaturaDev'in “özel entegratör” veya “GİB'e doğrudan bağlı servis”
olarak yorumlanmasını önler.

## API dokümanı

Kanonik OpenAPI sözleşmesi yayımlandığında bu sitedeki API Referansı sekmesi:

- kullanılabilir endpoint ve request şemalarını,
- kimlik doğrulama yöntemini,
- hata ve idempotency sözleşmesini,
- desteklenen e-Belge ve sağlayıcı capability'lerini,
- test ve production ortam sınırlarını

gösterecektir. Bu bilgiler yayımlanana kadar endpoint veya payload tahmin
etmeyin.

<Note>
  Bir özel entegratörün test hesabı veya bağlantı koşulları için yalnız o
  sağlayıcının resmî ve güncel public kanallarını kullanın. FaturaDev'de bir
  sağlayıcı bağlantısı ancak public ürün dokümanında yayımlandığında
  kullanılabilir kabul edilir.
</Note>
