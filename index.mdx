## E-Belge entegrasyonunda iki ayrı sorumluluk alanı

FaturaDev, yazılım ekipleri için ortak bir e-Belge API yüzeyi sunar. Mali imza,
GİB iletimi ve özel entegratör yükümlülükleri ise müşterinin yetkilendirdiği özel
entegratörde kalır.

## Tek geliştirici yüzeyi, ayrılmış yasal sorumluluklar

Müşteri yazılımı FaturaDev API ile konuşur. FaturaDev, yalnız müşterinin
yetkilendirdiği özel entegratör bağlantısı üzerinden çalışır. FaturaDev özel
entegratör değildir, GİB'e doğrudan bağlanmaz ve mali mühür, e-imza veya XAdES
üretmez.

## Yayımlanmış sözleşmeyi esas alın

API operasyonları, desteklenen e-Belge türleri ve kullanılabilir sağlayıcı
bağlantıları yalnız public sözleşmeleri yayımlandığında bu sitede listelenir.

<Warning>
  Bu sitede API referansı bulunmayan bir endpoint, payload, sağlayıcı bağlantısı
  veya capability kullanılabilir ürün özelliği olarak değerlendirilmemelidir.
</Warning>
